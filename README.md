OpenSearch vs PostgreSQL: Сравнительный анализ для хранения данных ML-моделей.

Репозиторий содержит инфраструктуру кластерных решений PostgreSQL и OpenSearch для хранения данных ML-модели с цепочками процессов.

Описание проекта:
Этот проект реализует развертывание всей необходимой инфраструктуры для тестирования:
PostgreSQL 15 (кластер: мастер + 2 реплики)
OpenSearch 2.17.1 (кластер из 3 нод)

Цель — определить оптимальную инфраструктуру для хранения цепочек процессов.
