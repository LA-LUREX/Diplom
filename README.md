OpenSearch vs PostgreSQL: Сравнительный анализ для хранения данных ML-моделей.

Репозиторий содержит инфраструктуру кластерных решений PostgreSQL и OpenSearch для хранения данных ML-модели с цепочками процессов.

Описание проекта:
Этот проект реализует полный цикл нагрузочного тестирования двух архитектур хранилищ данных для результатов работы ML-моделей, а также развертывание всей необходимой инфраструктуры:
PostgreSQL 15 (кластер: мастер + 2 реплики)
OpenSearch 2.17.1 (кластер из 3 нод)

Тестирование проводилось на 3-х виртуальных машинах с Ubuntu 22.04 в среде VirtualBox. Цель — определить оптимальную инфраструктуру для хранения цепочек процессов.
